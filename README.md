# 🚗 Laravel CRUD Data Mobil Sederhana

Aplikasi web sederhana untuk mengelola data mobil menggunakan **Laravel 13** dengan fitur CRUD (Create, Read, Update, Delete).

## ✨ Fitur

- ✅ **Create** - Tambah data mobil baru
- 👁️ **Read** - Tampilkan semua data mobil dalam tabel
- ✏️ **Edit** - Ubah data mobil yang sudah ada
- 🗑️ **Delete** - Hapus data mobil dengan konfirmasi
- ✔️ **Validasi Form** - Validasi input otomatis
- 🔔 **Alert Notifikasi** - Notifikasi sukses saat menambah/edit/hapus data
- 🎨 **Tampilan Rapi** - UI sederhana dan responsif

## 🛠️ Teknologi yang Digunakan

- **Backend**: PHP 8.x + Laravel 13
- **Database**: MySQL
- **Frontend**: Blade Template
- **Server**: Apache/Nginx (XAMPP/Laragon)

## 📋 Persyaratan Sistem

- PHP >= 8.0
- Composer
- MySQL/MariaDB
- Web Server (Apache/Nginx)
- XAMPP/Laragon (direkomendasikan untuk development)

## Cara Setup Database

Import File SQL (Paling Mudah)

1. Buka **phpMyAdmin**
2. Buat database baru bernama: `rozy`
3. Klik database `rozy`
4. Pilih tab **Import**
5. Pilih file `database/sql/rozy.sql`
6. Klik **Go** atau **Kirim**
7. Selesai! Tabel sudah terbuat
